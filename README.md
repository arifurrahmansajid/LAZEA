🌿 LAZEA – Plant-Based Website
A modern and responsive web application dedicated to promoting a plant-based lifestyle. LAZEA allows users to explore a wide variety of plant-based items, read detailed descriptions, and enjoy a seamless user experience designed to inspire green living.

🔍 Features
Explore Plant Items: Browse a collection of plant-based products and ingredients with key information.

Category Navigation: Filter and view plants by specific categories (Herbs, Vegetables, Indoor Plants, etc.).

Responsive Design: Fully responsive and mobile-friendly UI using Tailwind CSS.

Interactive Slider: Engaging homepage banner created using Swiper.js to showcase highlights and offers.

User Authentication: Secure login and registration system powered by Firebase.

Dynamic Details View: Click on any item to view detailed information including benefits and usage.

🛠️ Technologies Used
Frontend:
React JS – Component-based architecture for building the UI.

Tailwind CSS – Utility-first CSS framework for responsive design.

Swiper.js – For smooth, touch-enabled carousels and sliders.

Backend:
(Optional if added) Express.js, MongoDB – For data management and API (if backend is implemented).

Authentication:
Firebase Authentication – Secure and easy-to-integrate user authentication system.
